Features

Function emmet() have only two arguments, and it's enough to make awesome!

  • 1 - First argument - emmet input
  • Just write emmet('h1.repo-name{Emmet.js}+p.repo-description{Minimize your code with emmet.js}');
  • 2 - Second argument - object
  • Use object to build some information dynamically: emmet('h1.repo-name{%name%}+p.%class%{Minimize your code with emmet.js}',{name:"Emmet.js",class:"repo-description"});
